# RKDS Dental Clinic Manager

### Building from source & Deployment

After cloning the repository, run `npm install`, this will install the required dependencies. Then run `npm run prod` to build for production, or `npm run dev` to build for development.

### Scripts

-   `npm run prod`: build for production.
-   `npm run dev`: build for development.
-   `npm run lint`: run eslint on the project.
-   `npm run serve`: run a localhost server that serves the files in `dist/application/`.
-   `npm run build-desktop`: will create a desktop electron application.
-   `npm run deploy`: will build the application in production mode and then upload it.
-   `npm run jest`: will run unit tests.
-   `npm run cypress`: will run e2e tests.

### Documentation

-   See the `docs/` folder or open `index.html` in the project root.
